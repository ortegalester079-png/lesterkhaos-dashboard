# @lesterkhaos · Content OS

Tablero de contenido para la marca personal **@lesterkhaos** — psicología
humana, ego, trauma, identidad, sombra, relaciones, conciencia y transformación
interna. Pensado para usar el contenido como herramienta de **posicionamiento**,
**diagnóstico psicológico** y **conversión a conversación privada**.

Stack: **Next.js 14 · TypeScript · Tailwind CSS · shadcn/ui · Zustand**.
Modo oscuro, paleta carbón + terracota.

## Arranque rápido

```bash
npm install
npm run dev
# abre http://localhost:3000
```

La app arranca con datos ficticios (mock) ya cargados, así que puedes ver todo
funcionando desde el primer momento. Los datos se guardan en `localStorage`.

## Módulos

1. **Baúl de Ganchos** — guarda, clasifica y reutiliza hooks.
2. **Métricas** — rendimiento a 7/30/90 días, "bombazos" y top 5.
3. **Rastreador de Competencia** — análisis manual de cuentas referentes.
4. **Community Manager** — copy multiplataforma desde una sola pieza.
5. **Calendario de Contenido** — vistas mensual/semanal/diaria con drag & drop.
6. **Tendencias** — resumen diario con potencial de gancho.
7. **Generador de Guiones** — hooks → guiones con la voz de marca.
8. **Biblioteca de Ideas** — captura cruda antes de producir.
9. **Diagnóstico Semanal** — qué funcionó, qué no y qué probar.
